# InstitutePro App

This is the mobile application for the InstitutePro system, built with React Native and Expo.

## Project Structure

- `src/screens`: Contains all the screen components organized by feature.
- `src/components`: Reusable UI components.
- `src/navigation`: Navigation configuration (Stack, Bottom Tabs).
- `src/services`: API service calls.
- `src/context`: React Context for global state.
- `src/hooks`: Custom React hooks.
- `src/utils`: Utility functions and constants.
- `src/assets`: Images, fonts, and animations.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the app:
   ```bash
   npx expo start
   ```

## Key Dependencies
- React Navigation
- Axios
- Expo Secure Store
- Lottie React Native
